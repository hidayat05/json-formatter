# JSON Formatter

A web-based GUI application built with Rust and Yew for JSON manipulation.

## Features

- **Minify JSON**: Remove all unnecessary whitespace from JSON
- **Format JSON**: Pretty-print JSON with proper indentation
- **JSON to String**: Convert JSON to an escaped string literal
- **String to JSON**: Parse escaped JSON string literals back to formatted JSON
- **Copy Output**: One-click copy output to clipboard with the ðŸ“‹ Copy button
- **Copy Input**: One-click copy input to clipboard with the ðŸ“‹ Copy button
- **Clear All**: Reset both input and output fields

## Requirements

- Rust 1.88+ (currently using 1.91.1)
- Trunk (WASM web bundler): `cargo install trunk`
- wasm32-unknown-unknown target: `rustup target add wasm32-unknown-unknown`

## Building

```bash
# Install prerequisites (if not already installed)
rustup target add wasm32-unknown-unknown
cargo install trunk

# Development build and serve with hot reload
trunk serve

# Production build
trunk build --release
```

## Running

### Development Mode
```bash
trunk serve
```
This will:
- Build the app in development mode
- Start a local server at http://localhost:8080
- Open your browser automatically
- Watch for file changes and auto-reload

### Production Build
```bash
trunk build --release
```
The built files will be in the `dist/` directory. You can serve them with any static file server:
```bash
# Using Python
python3 -m http.server 8000 --directory dist

# Using a simple static server
cd dist && npx serve
```

## Usage

1. Open http://localhost:8080 in your browser
2. Enter or paste JSON content into the **Input** area (fully editable)
3. Click one of the action buttons:
   - **Minify JSON**: Compresses JSON to a single line
   - **Format JSON**: Formats JSON with indentation
   - **JSON â†’ String**: Converts JSON to an escaped string
   - **String â†’ JSON**: Converts an escaped string back to JSON
4. View the result in the **Output** area
5. Click the **ðŸ“‹ Copy** button to copy the input or output to your clipboard
6. Status messages appear at the bottom (green for success, red for errors)

## Examples

### Minify JSON
**Input:**
```json
{
  "name": "John",
  "age": 30
}
```
**Output:**
```json
{"name":"John","age":30}
```

### Format JSON
**Input:**
```json
{"name":"John","age":30}
```
**Output:**
```json
{
  "name": "John",
  "age": 30
}
```

### JSON to String
**Input:**
```json
{"name":"John"}
```
**Output:**
```
"{\"name\":\"John\"}"
```

### String to JSON
**Input:**
```
"{\"name\":\"John\"}"
```
**Output:**
```json
{
  "name": "John"
}
```

- **Trunk**: WASM web application bundler
